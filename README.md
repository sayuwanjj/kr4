Описание

Приложение позволяет пользователю: просматривать список технологий; добавлять новые технологии и удалять старые; отмечать прогресс изучения; оставлять заметки; импортировать дорожные карты; экспортировать дорожные карты; фильтровать и управлять технологическим стеком; производить поиск карточек.

##Установка:

1) Установите удалённый репозиторий 
2) Пропишите в терминал в папке проекта `npm install` и `npm start`.
3) Должно быть по адресу http://localhost:3000/
